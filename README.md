# SmartBusAssistant
**Purpose:** Developed an accessibility-focused Android application that enables visually and hearing-impaired users to independently track buses, navigate routes, and receive real-time travel assistance through voice, GPS, and smart alerts. 🚍📍♿

## Features

* Developed **Smart Bus Assist**, an Android application using **Kotlin** and **Jetpack Compose** to help visually and hearing-impaired users navigate public transportation independently.

* Implemented **voice-controlled bus search** using Android **SpeechRecognizer** and **Text-to-Speech (TTS)** APIs, achieving approximately **90% voice recognition accuracy**.

* Built a **real-time GPS tracking system** with **FusedLocationProviderClient**, providing location accuracy within **±10-15 meters** through distance-averaging algorithms.

* Integrated **Google ML Kit Face Detection** to estimate crowd density and classify environments as **Safe, Moderate, or Crowded** with around **85% accuracy**.

* Engineered **Firebase Realtime Database** for live synchronization between passenger and bus devices, delivering **voice, vibration, and visual alerts** for enhanced accessibility.
